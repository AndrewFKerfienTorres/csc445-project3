# BlackJack

> A simple game where the objective is to have the greatest score not above 21.
> The values of the cards are the numbers printed on them, except the Jack, Queen and King, which all
> are valued in 10. The ace may be either a one or an eleven.
> An ace and any 10-point card as a combo is called a 'BlackJack'. this combo is 11 + 10, so from 0 points this means 21 points -> instant win.
